# Capturas de Pantalla

Esta carpeta contiene las capturas de pantalla del proyecto para el README principal.

## Estructura recomendada

Coloca las siguientes capturas de pantalla con estos nombres:

- `login.png` - Página de inicio de sesión
- `dashboard.png` - Dashboard principal con estadísticas
- `inventario.png` - Vista del inventario
- `prestamos.png` - Sistema de préstamos
- `usuarios.png` - Gestión de usuarios
- `reportes.png` - Sistema de reportes
- `configuracion.png` - Panel de configuración

## Formato recomendado

- **Formato**: PNG o JPG
- **Resolución**: Mínimo 1920x1080px (o proporción similar)
- **Tamaño**: Optimizar para web (< 500KB por imagen)
- **Estilo**: Capturas completas de la pantalla o secciones destacadas

## Cómo tomar las capturas

1. Abre el proyecto en tu navegador
2. Navega a cada sección
3. Toma capturas de pantalla usando:
   - Windows: `Win + Shift + S` o `Print Screen`
   - Mac: `Cmd + Shift + 4`
   - Linux: `Print Screen` o herramientas como `gnome-screenshot`
4. Guarda las imágenes en esta carpeta con los nombres sugeridos
5. Actualiza las rutas en el README principal si usas nombres diferentes
